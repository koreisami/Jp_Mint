Linux Mint 17.2 "Rafaela" Cinnamon RC 版がリリースされました。

開発チームは Linux Mint 17.2 “Rafaela” Cinnamon RC 版のリリースについて発表します!

Linux Mint 17.2 Rafaela Cinnamon Edition

Linux Mint 17.2 は長期サポートリリースで、2019 年までサポートされます。 このバージョンでは快適に使用していただけるために、多くの更新や改善を受けることができます。

一目で分かる新機能:

  • Cinnamon 2.6
  • ソフトウェアソース
  • アップデートマネージャ
  • 言語設定
  • ログイン画面
  • システムの向上
  • アートワークの向上
  • その他の改善
  • 主な構成部分
  • LTS 戦略

全ての概略と新しい機能のスクリーンショットを見るには、以下のページをご 覧ください: “What’s new in Linux Mint 17.2 Cinnamon“.

重要な情報:

  • Flash の DRM コンテンツの視聴について
  • スカイプについての問題
  • VLC のDVD プレイバック
  • EFI サポート
  • ホームディレクトリの同期におけるスワップ領域に関する設定失敗について
  • NVIDIA GeForce GPUsの環境で固まってしまう件の解決
  • non-PAE のCPUでのブート
  • KDE アプリに関する問題
  • その他の問題

このリリースに関する重要な情報や既知の問題を把握するために “Release Notes” を必ず見てください。

システム要件:

  • x86 プロセッサ (Linux Mint 64-bit には 64 ビットプロセッサが必要です。 Linux Mint 32-bit は 32 ビットと 64 ビットプロセッサの両方で動きます)
  • 512 MB RAM (快適に使用するためには1 GB 以上が推奨)
  • 5 ギガバイトの空きディスク容量 (20GB 推奨)
  • 800×600 の解像度を表示できるグラフィックカード (1024×768 推奨)
  • CD/DVD ドライブ又は USB ポート

バグ報告:
