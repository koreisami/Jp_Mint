Linux Mint 19 "Tara" MATE がリリースされました!

開発チームは Linux Mint 19 “Tara” MATE のベータ版のリリースについて発表します!

Linux Mint 19 Tara MATE エディション

Linux Mint 19 は長期リリースバージョンで2023年までサポートされます。サポート期間内は、ソフトウェアの更新、改良、新しいソフトウェアが提供されます。そして、より良いデスクトップ環境を使用していただけると思います。

新機能:

Linux Mint の新バージョンは多くの改善が含まれています。

次のページを読むことで、新機能の要約を知る事ができるでしょう:

“What’s new in Linux Mint 19 MATE“.

重要な情報:

既知の問題の説明、回避方法、解決方法についてはリリースノートに記されています。

次のページよりリリースノートを読んでください:

“Release Notes for Linux Mint 19 MATE”

システム要件:

  • 1GB RAM (満足な使用のためには、メモリが2GB以上が推奨されます).
  • 15ギガバイトの空きディスク容量 (20GB 推奨)
  • 1024×768 解像度。 (低解像度でもしスクリーンに合わない場合は、マウスでALTを押しながらウインドウをドラッグさせてください).

ノート:

  • 64ビット版 ISO イメージは BIOS または UEFIからのブートが可能です。
  • 32ビット版 ISO イメージは BIOS からのブートのみ可能です。
  • 現役の主なるコンピュータで使用される場合は 64 ビット版のイメージの使用が推奨されます (2007年以降に発売されたたいていのコンピュータです).

アップグレードに関する情報

Linux Mint 18.3 からの更新方法の手順がアナウンスされています。

もし、ベータバージョンを使用し、システムスナップショットを使用し、アップデートマネージャで更新を行う場合、次のコマンドを実行し、システムの再起動を行ってください。:

apt remove ttf-mscorefonts-installer
apt install libreoffice-sdbc-hsqldb sessioninstaller ttf-mscorefonts-installer
sudo rm -f /etc/systemd/logind.conf
apt install --reinstall -o Dpkg::Options::="--force-confmiss" systemd
sudo rm -f /etc/polkit-1/localauthority/50-local.d/com.ubuntu.enable-hibernate.pkla

完全性と信頼性のチェック:

ダウンロードが完了しましたら、完全性と信頼性の検証を行ってください。

誰でも偽造のISO イメージが作成できてしまうため、公式のISO イメージをダウンロードしたかどうかのチェックは、あなた自身で行う必要があります。

  • 実施は次のページのステップに従ってください。 https://linuxmint.com/verify.php
  • チェックサム: sha256sum.txt
  • 署名されたチェックサム: sha256sum.txt.gpg
